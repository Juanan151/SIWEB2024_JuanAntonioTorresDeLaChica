# ENTREVISTA

1. ¿Tienen los lectores, los posibles clientes, interés en los sitios web de las editoriales?

Sí, definitivamente hay un interés. Los lectores suelen buscar sitios web de editoriales para obtener información detallada sobre libros, autores, lanzamientos futuros y eventos. Estos sitios sirven como un punto de contacto directo entre el editor y su audiencia.

  2. ¿Por qúe puede ser esto?

Los sitios web de editoriales pueden ofrecer contenido exclusivo, como entrevistas con autores, capítulos de muestra, y detalles sobre la creación de sus obras. Además, para los lectores apasionados por ciertos géneros o autores, estos sitios web ofrecen una forma de seguir de cerca a sus favoritos.

  3. ¿Son los libros de las editoriales con buen sitio web más vendidos?
     
No siempre los libros de editoriales con buenos sitios web son los más vendidos, pero un sitio web atractivo y fácil de usar puede mejorar la visibilidad de los libros y, por ende, potencialmente aumentar las ventas. Un buen sitio web puede ser un factor importante en la decisión de compra de un lector.

  4. ¿Dónde suelen buscar el contacto de las editoriales, las distribuidoras?
     
Los lectores y profesionales del libro suelen buscar el contacto de editoriales en sus sitios web oficiales. Estos contactos son importantes para consultas, servicios de prensa, o incluso para enviar manuscritos.

  5. ¿Qúe diferencias hay sobre la información, que conocen los lectores, de libros con buen sitio web o mal sitio web?
      
Los sitios web bien diseñados tienden a ofrecer información más clara, detallada y actualizada sobre los libros y autores. Un sitio web mal diseñado o desactualizado puede resultar en una falta de información o en la presentación de datos incorrectos, afectando la percepción del lector sobre la editorial.

  6. ¿Qué apartados de estos sitios web son más interesantes?
      
Los lectores valoran apartados como próximos lanzamientos, catálogos completos de publicaciones, biografías de autores, blogs con artículos relacionados con el mundo editorial y la literatura, y secciones de contacto o soporte.

  7. ¿En qué crees que afecta tener sitio web?
      
Tener un sitio web permite a las editoriales construir una presencia de marca en línea, interactuar directamente con su audiencia y proporcionar un canal adicional para la promoción y venta de libros. Es una herramienta esencial para la visibilidad en el mercado actual.

  8. ¿Qué información se espera encontrar al entrar en estos?
      
Los usuarios esperan encontrar información detallada sobre los libros (sinopsis, autor, fecha de publicación, género), noticias de la editorial, eventos, políticas de envío y devolución para compras, y opciones de contacto. También valoran contenidos adicionales como blogs, entrevistas, y acceso a material exclusivo.
